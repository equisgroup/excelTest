# Excel Dashboard Generator

Generador de Excel con Dashboard Gerencial y Control de Asignaciones de Empleados.

## Descripción

Este proyecto es una solución .NET que genera archivos Excel interactivos con dashboards, slicers y sistema de alertas para gestión de empleados, clientes, asignaciones, vacaciones, viajes y turnos de soporte.
